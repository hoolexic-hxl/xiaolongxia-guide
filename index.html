详细步骤

每一步都配有「避坑提醒」，新手也能一次成功

1
准备工作

购买腾讯云服务器

腾讯云轻量应用服务器 99 元/年，2核2G 足够跑小龙虾。

  • 访问腾讯云官网，搜索「轻量应用服务器」
  • 选 99 元/年入门套餐（2核2G + 40G SSD）
  • 创建方式选「应用模板」，镜像选「OpenClaw(龙虾)」
  • 设好密码，选地域，付款
# ⚠️ 关键：必须选「应用模板」里的 OpenClaw # 选错成系统镜像，装出来是空 Linux，还得手动装
2
核心操作

确认 OpenClaw 已安装

买的时候选了应用模板，一般已经装好了。只需确认一下。

  • 进「轻量应用服务器控制台」
  • 看服务器 Logo 是不是 🦞
  • 是 → 跳过这步
  • 不是 → 点「重装系统」→ 选「应用模板」→「OpenClaw」→ 确认
# 重装约 1 分钟，期间会重启，正常
3
赋予智能

配置 DeepSeek 大模型

小龙虾需要大脑。接入 DeepSeek，让它能思考能对话。

  • 点服务器名进详情页 →「应用管理」
  • 找到「自定义模型」，填 DeepSeek API 地址和 Key
  • 点「添加并应用」，等状态变「应用中」
  • 或直接用腾讯云内置 DeepSeek 通道，更省事
# 推荐配置： 模型: DeepSeek-R1 或 DeepSeek-V3 API: https://api.deepseek.com/v1 # 内置通道无需额外申请 Key
4
连接世界

接入聊天平台

让小龙虾住进你常用的聊天工具，随时@它。

  • 推荐：「应用管理」面板选平台（飞书/企微/QQ/钉钉），扫码授权
  • 进阶：终端执行 openclaw onboard 启动交互向导
  • 按引导选渠道 → 填 Token → 完成
  • 发一条测试消息，确认能回复
# 快速验证： openclaw tui # 打开对话界面 openclaw web # 启动 Web 后台（端口 18789）
5
安全加固

权限设置

给小龙虾划边界，让它听话不越权。

  • 工作区：只开指定文件夹，禁访问系统盘
  • 操作权限：关闭系统修改、软件安装、文件删除
  • 二次确认：任何删除/修改都需你点头
  • 记忆库：配置长期记忆，越用越懂你
# 原则：只给必要权限，不给多余自由 # 它是助手，不是管理员 😄

避坑指南 & 进阶技巧

前人踩过的坑，你就不要再踩了

🔥

防火墙记得放行

如果 Web 后台打不开，90% 是因为安全组没放行 18789 端口。去控制台 → 防火墙 → 添加规则，TCP 协议放行即可。

💸

模型费用控制

日常任务国产模型完全够用，月均 30~80 元。不要盲目上 GPT-4o 或 Claude，新手 90% 的场景用不上。

🔑

API Key 保管

Key 即密码，不要上传到 GitHub 或公开平台。建议单独保存在本地密码管理器，定期轮换。

🐳

本地部署备选

如果云服务器到期，可以用 Docker 本地部署：docker compose up -d，数据全在本地，隐私性更强。

🛠

Skills 技能商店

OpenClaw 支持插件扩展。后续可以在 Skill 商店安装「会议纪要」「竞品监控」「社群管理」等实用工具。
